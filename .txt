Папка logistics (додаток):
    admin.py: Налаштування адмін-панелі для моделей додатку.
    apps.py: Конфігурація самого додатку.
    models.py: Визначення моделей бази даних.
    serializers.py: Серіалізатори для перетворення даних у формат JSON (використовується для API).
    views.py: Логіка обробки запитів (контролери).
    urls.py: Маршрути (URL-адреси) для цього додатку.
    utils.py: Допоміжні функції для цього додатку.
    signals.py: Сигнали Django для обробки подій (наприклад, після збереження моделі).
    migrations/: Папка для міграцій бази даних.

Папка transport_project (основний проект):
    __init__.py: Ініціалізує цю папку як Python-пакет.
    asgi.py: ASGI-файл для запуску асинхронного сервера.
    settings.py: Основний файл з налаштуваннями проекту (база даних, середовище тощо).
    urls.py: Головний файл маршрутизації для всього проекту.
    wsgi.py: WSGI-файл для запуску синхронного сервера.
    .gitignore: Файл для ігнорування певних файлів/папок у Git.
    db.sqlite3: Локальна база даних SQLite для розробки.
    manage.py: Основна утиліта для керування проектом (запуск сервера, міграції тощо).
    package-lock.json: Файл з залежностями для JavaScript (можливо, це фронтенд).
    setup_env.bat: Скрипт для налаштування середовища розробки (для Windows).
    
Інше:
    _pycache__/: Папка для кешування скомпільованих Python-файлів.
